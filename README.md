# ❧ Sayap-Sayap Patah — Digital Library
Repositori ini berisi publikasi web statis dari mahakarya Kahlil Gibran, **"Sayap-Sayap Patah"**, yang diterjemahkan secara puitis ke dalam Bahasa Indonesia.

## ✨ Fitur Utama
* **Desain Klasik:** Tipografi menggunakan *Playfair Display* untuk nuansa sastra yang kental.
* **Antarmuka Perkamen:** Visual bertema kertas tua (parchment) yang nyaman dibaca.
* **Navigasi Presisi:** Navigasi per bab yang memudahkan pembaca menjelajahi kisah Selma Karamy.
* **Responsive:** Optimal dibaca melalui perangkat mobile maupun desktop.

## 🛠️ Teknologi
* HTML5 / CSS3
* Google Fonts API
* GitHub Pages (Hosting)
